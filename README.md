# Text Adapter

Chrome-расширение для упрощения сложного текста на веб-страницах с помощью AI (Groq). Результат показывается прямо на странице — оригинал не перезаписывается.

## Установка

1. Скачайте или клонируйте репозиторий.
2. Откройте Chrome → `chrome://extensions/`.
3. Включите **Режим разработчика**.
4. **Загрузить распакованное** → выберите папку `extension/`.
5. Закрепите иконку Text Adapter на панели Chrome.

### Первоначальная настройка

1. Кликните правой кнопкой по иконке → **Параметры** (Options).
2. Зарегистрируйтесь на [console.groq.com](https://console.groq.com) и создайте **API key**.
3. Вставьте ключ → **Save** → **Test connection** (должно быть OK).

Ключ хранится только в вашем браузере и не попадает в репозиторий.

## Как пользоваться

1. Откройте обычный сайт (статья, документация, Wikipedia).
2. Выберите режим:
   - **Выделение** — сначала выделите текст мышью, затем нажмите иконку расширения.
   - **Вся страница** — адаптируется основной контент статьи.
3. Выберите уровень: **Light**, **Medium** или **Max**.
4. Нажмите **Адаптировать** → дождитесь «Готово».
5. На странице появится блок с кнопками:
   - **Адаптировано** — упрощённый текст
   - **Оригинал** — исходный текст
   - **Сравнение** — два столбца рядом
   - **Вернуть как было** — убрать блок, страница как раньше

## Уровни упрощения

| Уровень | Что делает | Где лучше применять |
|---------|------------|---------------------|
| **Light** | Лёгкая правка: чуть проще фразы, длинные предложения разбиваются. Термины и факты сохраняются, длина почти как в оригинале. | Если нужно сохранить терминологию и факты (научные тексты, какая-то документациия). |
| **Medium** | Умеренное упрощение: понятнее формулировки, 1–2 сложных термина кратко поясняются. Все факты на месте. | Универсальный вариант: Wikipedia, документация, научно-популярные статьи. Хороший выбор по умолчанию. |
| **Max** | Максимально простые слова и короткие предложения, текст короче оригинала, без «воды». Факты не выбрасываются. | Юридические тексты, медицина, техническая документация — когда нужно «объяснить простыми словами». |

## Структура проекта

```
extension/
  manifest.json        — манифест расширения
  content_script.js    — чтение текста и UI на странице
  service_worker.js    — запросы к Groq API
  popup.html/js        — окно управления
  options.html/js      — настройки (API key)
```
